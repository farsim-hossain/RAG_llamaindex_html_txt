Waldorf 365 Cloud

Häufig gestellte Fragen

Stichwörter: FAQ

Zeugnisdaten müssen ja für 10 Jahre aufbewahrt werden. Wie wird dies durch Waldorf 365 unterstützt?

Wir unterstützen die Aufbewahrung der Zeugnisse in mehrerer Hinsicht:

  • Prinzipiell können auf dem Server beliebig viele Schuljahre gespeichert werden. Somit speichern bleiben alle Daten (also insbesondere die Zeugnisdaten) für die letzten 10 Jahre erhalten. Dies ist auch bei Stundenplänen etc. immer wieder wichtig einen Vergleich mit alten Daten herstellen zu können.
  • Der automatische Backup-Mechanismus erstellt täglich eine Sicherheitskopie (von allen Schuljahren) und speichert 30 solche Sicherheitskopien (für die letzten 30 Tage) auf einem separaten System. Im Falle eines Totalausfalls des Servers mit Datenverlust (sehr unwahrscheinlich) können die Daten über das Backupsystem wiederhergestellt werden.
  • Das Zeugnissystem bietet darüberhinaus über ein paar Klicks die Möglichkeit, am Ende eines Schuljahres, die Zeugnisdaten als Archiv zu exportieren. Wir empfehlen als zusätzliche Sicherheit diese Archivdatei auf einem USB-Stick oder einer DVD abzuspeichern und für 10 Jahre vorzuhalten. Dies hat den Vorteil, dass Sie nicht auf unser System angewiesen sind, um die Daten vorzuhalten. Dazu werden die Daten in drei verschiedenen Formaten exportiert.
    • Das PDF-Format garantiert, dass die Zeugnisdaten unverändert nachgedruckt werden können. Dieses Format speichert das exakte Druckbild ab.
    • Das ODT-Format kann mit jeder Textverarbeitung geöffnet werden und ermöglicht die weiterverwendung aller Daten mit jedem beliebigen anderen Softwaresystem. Sie bleiben also wirklich von Waldorf 365 unabhängig.
    • Das LaTeX-Format wiederum ist ein Zwischending zwischen den beiden anderen Formaten. Es ermöglicht eine Weiterverwendung und Weterverarbeitung der Daten allerdings indem die exakte Formatierung der Daten beibehalten wird.

Der Stundenplan fürs kommende Schuljahr ist nicht auf der Cloud sichtbar. Was mache ich falsch?

  • Stellen Sie sicher, dass das Anfangsdatum und Enddatum des Stundenplans stimmen (diese Daten müssen das aktuelle Datum umfassen).
  • Stellen Sie sicher, dass das Schuljahr des Stundenplanes auf der Cloud aktiv ist. Klicken Sie dazu im Waldorf 365 Planer auf Grunddaten > Schuljahre > Spalte: Auf Cloud aktiv. Es genügt hier nicht, nur die Daten des Schuljahres richtig einzustellen, so dass sie das aktuelle Datum umfassen, sondern Sie müssen zusätzlich in der Spalte Auf Cloud aktiv beim kommenden Schuljahr auf aktivieren klicken.
  • Stellen Sie sicher, dass das Anfangsdatum und Enddatum des Stundenplans stimmen (diese Daten müssen den aktuellen Tag umfassen).

Auf der Cloud werden alle Stunden doppelt angezeigt. Warum?

Auf der Cloud werden alle Stundenpläne angezeigt, deren Gültigkeitszeitraum das aktuelle Datum umfassen. Doppelte Stunden entstehen, wenn aus Versehen ein Stundenplan-Entwurf Anfangs- und Enddaten hat, so dass er am aktuellen Tag zusammen mit dem eigentlichen Stundenplan gültig ist.

Wie ist der Datenschutz und Zugriff auf Zeugnisdaten geregelt?

Prinzipiell erfolgt der Zugriff auf Waldorf 365 über einen SSL-verschlüsselten und passwortgeschützten Zugang per Internet-Browser oder Smartphone-App. Es gibt verschiedene Rollen und Rechte, z. B. Eltern/Schüler, Mitarbeiter, Lehrer, Zeugnis-Administrator, Klassenbuch-Administrator etc. (also für jedes Modul eine spezifische Administrator-Rolle) und die generelle Administrator-Rolle. Die Lehrer-Rolle ermöglicht Schreib-Zugriff auf die eigenen zu schreibenden Zeugnisse, das gegenseitige Korrigieren der Zeugnisse während eines aktiven Zeugnisprozesses (Der Korrekturvorschlag wird dem Schreibenden wieder vorgelegt) und das Lesen der Schülerzeugnisse aller Schüler. Außerdem ermöglicht es den Lese-Zugriff auf die in den letzten Jahren selber geschriebenen Zeugnisse. Bei der Erstkorrektur können alle Lehrer für die Zeugnisse der anderen Lehrer Korrekturvorschläge machen. Die Zweitkorrektur wird durch ein Rechtschreibprogramm vorsortiert und kann im Regelfall von allen Lehrern mit der Zeugnis-Administrator-Rolle durchgeführt werden. Zusätzlich lässt sich diese Funktion in den Einstellungen für alle Lehrer freischalten. Diese Funktion ermöglicht, die Zeugnisse zu ändern, ohne dass diese den Schreibenden erneut zur Korrektur vorgelegt werden. Sie ist dazu gedacht, offensichtliche Rechtschreibfehler, die bei der Erstkorrektur übersehen wurden, schnell zu finden und zu verbessern. In der Benutzerverwaltung können Personen mit der generellen Administrator-Rolle Benutzerkonten erstellen, löschen, die Rollen zuweisen und das Passwort neu generieren.

In welcher Reihenfolge werden die Vertreungslehrer angezeigt?

Die Lehrer in der Liste der möglichen Vertretungen werden nach Relevanz sortiert. Oben stehen also diejenigen, welche am ehesten für die Vertretung in Frage kommen, weil sie z. B. Vertretungsbereitschaft haben und unten stehen Lehrer, welche auf keinen Fall in Frage kommen, weil sie eine andere Veranstaltung / Unterricht haben. Innerhalb einer Relevanz-Kategorie sind die Lehrer alphabetisch sortiert, d. h. alle Lehrer, die eine Hohlstunde oder gleichwertiges haben, stehen in einem Block untereinander und sind innerhalb dieses Blocks alphabetisch sortiert. Diese Sortierung ist ungeschickt, wenn man einen ganz bestimmten Lehrer eintragen möchte. Um einen ganz bestimmten Lehrer einzutragen, kann man über der Liste der möglichen Vertretungslehrer auf „Sortieren“ klicken und dann „Alphabetisch sortieren“. Dann werden die Lehrer in alphabetischer Reihenfolge angezeigt.

Ein Lehrer wird in der Liste der möglichen Vertretungslehrer nicht angezeigt. Warum?

In der Liste der möglichen Vertretungslehrer werden nur Lehrer angezeigt, die mindestens einen Kurs unterrichten. Außerdem werden Lehrer nicht angezeigt, die mit dem Spezialfall „Lehrer macht keine Vertretungen“ markiert sind. Manchmal findet man einen Lehrer nicht auf Anhieb, weil die Liste standardgemäß nicht alphabetisch sortiert ist. In diesem Fall kann man über der Liste der möglichen Vertretungslehrer auf „Sortieren“ klicken und dann „Alphabetisch sortieren“.

Ich habe mit der Demoversion Daten erstellt. Kann ich diese nach Bestellung einer Vollversion übernehmen?

Dateien, welche mit der Demoversion erstellt wurden, können mit der Vollversion weiter verwendet werden.

Wenn Sie sich für einen Cloud-Server entscheiden, müssen Sie die vorhanden Daten zunächst auf den Server übertragen.

  • Achten Sie darauf, dass die Produktlizenz eingegeben wurde (Extras > Produktlizenz), so dass Sie über eine Vollversion des Planers verfügen.
  • Achten Sie darauf, dass das Feld unter Bearbeiten > Einstellungen > Server > Server-Adresse leer ist.
  • Laden Sie nun die erstellte Datei.
  • Klicken Sie dann auf Import/Export und anschließend Datei auf Cloud übertragen.

Dadurch wird die Datei auf den Server geladen.

Wichtig: Beenden Sie jetzt das Programm und starten Sie es neu.

Ab jetzt können Sie immer mit der zentralen Datei auf dem Server arbeiten.

Dazu wählen Sie beim Programmstart immer die Option Mit Cloud verbinden.
